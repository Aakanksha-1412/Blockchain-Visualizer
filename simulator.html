The Blockchain Simulator Lab

A hands-on environment to test, attack, and rebuild a local blockchain database. Edit details, trace mathematical proof invalidations, and execute custom consensus healing loops.

Interactive Sandbox Instructions

This blockchain is initially perfectly mined and valid. Type inside any block's Transaction Data field to alter records. Observe how the change instantly invalidates both that block and all following blocks. Use the individual Mine Block buttons to re-solve that block's mathematical difficulty target, or click the global Heal Chain button to automate the sequential mining loop!
